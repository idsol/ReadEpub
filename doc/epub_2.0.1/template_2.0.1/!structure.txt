-------------------------------------------------------------------
① /mimetype
-------------------------------------------------------------------
路径固定、内容固定。
内容总是“application/epub+zip”。

-------------------------------------------------------------------
② /META-INF/container.xml
-------------------------------------------------------------------
路径固定。
其中<rootfile full-path="content.opf">指向根文件 ③。
没有必要可以不作改动。
Open Container Format

-------------------------------------------------------------------
③ /content.opf
-------------------------------------------------------------------
文件名称路径可以修改，但要与 ② 中指向的路径保持一致。
包含书籍的元数据、文件顺序、文件组成、目录、封面页面等信息。
需要大量修改。
OPF = Open Packaging Format

-------------------------------------------------------------------
④ /toc.ncx
-------------------------------------------------------------------
NCX = Navigation Control file for XML applications

-------------------------------------------------------------------
⑤ /OPS/
-------------------------------------------------------------------
包含所有内容页面、样式表、图片等。
目录可以随意组织，但要与 ③ 的内容保持一致。
OPS = Open Publication Structure
